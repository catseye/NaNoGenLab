infix-neologisms
================

Requirements
------------

*   Python 2.7.6 (probably works with older versions too)
*   A set of input words

Basic Strategy
--------------

*   Pick an input word.
*   Split it into two parts.  Pick another input word and insert it in
    the middle.
*   Possibly repeat step #2.
*   Output the word and repeat from step #1.
